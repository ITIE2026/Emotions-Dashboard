Impedance measurement

Methods:

  • clCResistance_GetCount - the number of channels
  • clCResistance_GetChannelName - the name of a channel
  • clCResistance_GetValue - the resistance of a channel

The resistance value for each channel, in order to receive good signal, it is recommended to be lowered according to the following estimate:

  • Above 1000 kOhm - bad quality
  • 500 to 1000 kOhm - average quality
  • Below 500 kOhm - good quality

Get channel info:

You can use these methods any time after creating the device.

  • clCDevice_GetChannelNames
  • clCDevice_ChannelNames_GetChannelsCount
  • clCDevice_ChannelNames_GetChannelIndexByName
  • clCDevice_ChannelNames_GetChannelNameByIndex

Device mode events:

  • clCDevice_SetOnModeSwitchedEvent

In signal mode received EEG data already free of miographic artifacts. If artifacts detected, invalid values are not sent via API.
