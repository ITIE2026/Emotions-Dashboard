RAW and RAW filtered signal

If you don't need to use metrics, you can work with raw signal.

  • To work with raw signal directly from the device (EEG bandwidth from 1 to 120 Hz), you should subscribe to the events that contain raw signal after device connection.
  • To work with raw signal directly from the device filtered with verified digital filters (effective EEG measurement bandwidth set from 4 to 35 Hz), you should connect to the device, select EEG signal mode and create a session with monopolar or bipolar channel mode.
  • To use classifiers, follow the standard flow described above.
