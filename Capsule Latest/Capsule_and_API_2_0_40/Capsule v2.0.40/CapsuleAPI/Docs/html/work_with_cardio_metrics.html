Work With Cardio Metrics

To get [Timestamp, HeartRate, StressIndex, KaplanIndex, HasArtifacts, SkinContact, MotionArtifacts, MetricsAvailable] a callback is used with a pointer to clCCardio_Data, which is set via clCCardio_SetOnIndexesUpdateEvent

Methods:

  • clCCardio_Create
  • clCCardio_CreateCalibrated

Сardio events:

  • clCCardio_SetOnIndexesUpdateEvent
  • clCCardio_SetOnCalibratedEvent
  • clCCardio_SetOnPPGDataEvent
  • The skinContact responsible for detecting the position of the device, whether it is worn on the head or not.
  • The motionArtifacts parameter influences the current ppg signal. If artifacts are detected, it is advisable to secure the device to the head.
  • The metricsAvailable parameter indicates that the calibration has been completed and the metrics can be used.

Get data readings from PPG event:

  • clCPPGTimedData_GetCount
  • clCPPGTimedData_GetValue
  • clCPPGTimedData_GetTimestampMilli
