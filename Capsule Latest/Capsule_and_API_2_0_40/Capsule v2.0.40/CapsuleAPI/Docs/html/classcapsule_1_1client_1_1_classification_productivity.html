Detailed Description

Interface to Productivity process.

Before calling any of the functions make sure that:

  • A valid Session created with Client::CreateSession is passed to the constructor.
  • Session is active.

Model might need to be trained before it is able to calculate user productivity state. And need to be calibration baselines

In order to train model you should:

  • Mark each state at least once, with duration no less than a time window
