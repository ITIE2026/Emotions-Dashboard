Calibration

To get [Timestamp, FailReason, IndividualFrequency, IndividualPeakFrequency, IndividualPeakFrequencyPower, IndividualPeakFrequencySuppression, IndividualBandwidth, IndividualNormalizedPower, LowerFrequency, UpperFrequency] a callback is used with a pointer to clCIndividualNFBData, which is set via clCNFBCalibrator_SetOnCalibratedEvent

Individual NFB Calibration Stage values:

  • 0 - stage_1
  • 1 - stage_2
  • 2 - stage_3
  • 3 - stage_4

FailReason values (if the calibration process failed you will receive the following responses):

  • 0 - Calibration succeeded
  • 1 - Too many artifacts
  • 2 - Individual alpha peak frequency matches one of the alpha range borders

Methods:

  • clCNFBCalibrator_CreateOrGet
  • clCNFBCalibrator_CalibrateIndividualNFB
  • clCNFBCalibrator_CalibrateIndividualNFBQuick
  • clCNFBCalibrator_ImportIndividualNFBData
  • clCNFBCalibrator_GetIndividualNFB
  • clCNFBCalibrator_IsCalibrated
  • clCNFBCalibrator_HasCalibrationFailed

Calibration events:

  • clCNFBCalibrator_SetOnCalibrationStageFinishedEvent
  • clCNFBCalibrator_SetOnCalibratedEvent

Method clCNFBCalibrator_CalibrateIndividualNFB requires 4 stages to train model:

  1. Closed eyes - 20 sec
  2. Open eyes - 20 sec
  3. Closed eyes - 20 sec
  4. Open eyes - 20 sec

Method clCNFBCalibrator_CalibrateIndividualNFBQuick will require 1 stage to train model:

  1. Closed eyes - 30 sec
