Work With Accelerometer And Gyroscope Readings

This classifier combines the accelerometer and the gyroscope integrated in PSBD Headband Pro. It is called clCMEMS (MEMS - micro electro mechanical system) for brevity.

Methods:

  • clCMEMS_Create
  • clCMEMS_CreateCalibrated

MEMS events:

MEMS event contains multiple sensor readings coupled with timepoints

  • clCMEMS_SetOnMEMSTimedDataUpdateEvent

Get accelerometer and gyroscope readings from MEMS event:

  • clCMEMSTimedData_GetCount - the number of readings
  • clCMEMSTimedData_GetAccelerometer - accelerometer orientation
  • clCMEMSTimedData_GetGyroscope - gyroscope orientation
  • clCMEMSTimedData_GetTimestampMilli - timestamp of a reading
