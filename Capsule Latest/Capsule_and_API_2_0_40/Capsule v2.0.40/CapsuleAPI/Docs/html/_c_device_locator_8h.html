◆ clCCapsule_SetSingleThreaded()

CL_DLL void clCCapsule_SetSingleThreaded ( bool singleThreaded)

By default Capsule API operates in multi threaded mode - event callbacks are fired in a background thread. Call this function once before working with Capsule to initiate single-thread mode. In order for Capsule to work in this mode, clCDeviceLocator_Update must be called periodically on the main thread. The event callbacks will be called inside clCDeviceLocator_Update.
