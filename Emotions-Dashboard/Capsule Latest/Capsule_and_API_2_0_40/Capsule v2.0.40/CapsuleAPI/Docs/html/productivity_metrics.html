Productivity Metrics

To get [Timestamp, Gravity, Productivity, Fatigue, ReverseFatigue, Relax, Concentration] a callback is used with a pointer to clCProductivity_Baselines, which is set via clCProductivity_SetOnBaselineUpdateEvent

To get [Timestamp, Relaxation, Stress, GravityBaseline, ProductivityBaseline, FatigueBaseline, ReverseFatigueBaseline, RelaxationBaseline, ConcentrationBaseline, HasArtifacts] a callback is used with a pointer to clCProductivity_Indexes, which is set via clCProductivity_SetOnIndexesUpdateEvent

To get [Timestamp, FatigueScore, ReverseFatigueScore, GravityScore, RelaxationScore, ConcentrationScore, ProductivityScore, CurrentValue, Alpha, ProductivityBaseline, AccumulatedFatigue, FatigueGrowthRate, ArtifactsData, ArtifactsSize] a callback is used with a pointer to clCProductivity_Metrics, which is set via clCProductivity_SetOnMetricsUpdateEvent

The [currentValue, productivityScore] cannot be calculated until the calibration is completed.

To reset accumulated fatigue and start its calculation from zero method clCProductivity_ResetAccumulatedFatigue is used.

Recommendation salues:

  • 0 - no recommendation
  • 1 - involvement
  • 2 - relaxation
  • 3 - slight fatigue
  • 4 - severe fatigue
  • 5 - chronic fatigue

StressPoint values:

  • 0 - no stress
  • 1 - anxiety
  • 2 - stress

Fatigue Growth Rate values:

  • 0 - none
  • 1 - low
  • 2 - medium
  • 3 - high

Methods:

  • clCProductivity_Create
  • clCProductivity_CreateWithIndividualData
  • clCProductivity_ImportBaselines
  • clCProductivity_ResetAccumulatedFatigue
  • clCProductivity_StartBaselineCalibration

Initialize productivity events:

  • clCProductivity_SetOnBaselineUpdateEvent
  • clCProductivity_SetOnMetricsUpdateEvent
  • clCProductivity_SetOnIndexesUpdateEvent
  • clCProductivity_SetOnCalibrationProgressUpdateEvent
  • clCProductivity_SetOnIndividualNFBUpdateEvent
