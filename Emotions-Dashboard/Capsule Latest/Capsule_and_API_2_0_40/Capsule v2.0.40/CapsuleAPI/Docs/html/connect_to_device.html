Connect To Device

To get [DeviceID, IsConnected, SerialNum, PingTime, DeviceStatus, ChannelNames, ChannelNamesSize, Resistances, ResistancesSize, BatteryCharge] a callback is used with a pointer to clCDevice_StateData, which is set via clCDevice_SetOnDeviceStateUpdateEvent

To get clCDevice_ConnectionState a callback is used, which is set via clCDevice_SetOnConnectionStateChangedEvent

Fail Reason values:

  • 0 - Ok
  • 1 - BluetoothDisabled
  • 2 - Unknown

Device Type values:

  • 0 - Band
  • 1 - Buds
  • 2 - Headphones
  • 3 - Impulse
  • 4 - Any
  • 6 - BrainBit
  • 100 - SinWave
  • 101 - Noise

Device Mode values:

  • 0 - Resistance
  • 1 - Signal
  • 2 - SignalAndResist
  • 3 - StartMEMS
  • 4 - StopMEMS
  • 6 - StartPPG
  • 7 - StopPPG

Connection State values:

  • 0 - Disconnected
  • 1 - Connected
  • 2 - UnsupportedConnection

Log Level:

  • 0 - Trace
  • 1 - Debug
  • 2 - Info
  • 3 - Warn
  • 4 - Err
  • 5 - Critical
  • 6 - Off
