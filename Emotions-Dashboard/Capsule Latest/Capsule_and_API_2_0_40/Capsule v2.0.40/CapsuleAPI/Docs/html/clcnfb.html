clCNFB

To get [Timestamp, Delta, Theta, Alpha, SMR, Beta] a callback is used with a pointer to clCNFB_UserState, which is set via clCNFB_SetOnUserStateChangedEvent

Call Result values:

  • 0 - Success
  • 1 - FailedToSendData

Methods:

  • clCNFB_Create
  • clCNFB_CreateCalibrated

NFB events:

  • clCNFB_SetOnUserStateChangedEvent
  • clCNFB_SetOnErrorEvent

You can receive values of the following main brain rhythms without calibration: Theta (4-7 Hz), Alpha (7-13 Hz), Beta (15-30 Hz). All values are sent in mV^2.

Method clCNFB_CreateCalibrated allows you to create a classifier after calibration individual nfb data
